# Responsive_Portifolio
A fully responsive personal portfolio website built with HTML, CSS, and JavaScript to showcase my projects, skills, and professional profile. Designed with a clean, modern UI and optimized for all devices.

RESPONSIVE PERSONAL PORTFOLIO WEBSITE
 Author: Felix Haule
 Role: Computer Engineering Student | Junior Software Developer
 Year: 2026


PROJECT OVERVIEW

This is a fully responsive personal portfolio website developed to
showcase my professional profile, technical skills, services, and
completed projects.

The portfolio reflects my journey as a Computer Engineering student at
Dar es Salaam Institute of Technology (DIT). It combines engineering
precision with software development expertise to deliver practical and
impactful digital solutions.

The project demonstrates structured frontend development, clean user
interface design, responsive layout techniques, and integration of
modern JavaScript libraries.


KEY FEATURES

-   Fully responsive layout (Desktop, Tablet, Mobile)
-   Hero section with professional introduction
-   About section highlighting engineering and programming background
-   Services section with expandable content
-   Portfolio section displaying recent projects
-   Testimonial slider powered by Swiper.js
-   Contact form layout
-   Dark mode toggle feature
-   Scroll animations using ScrollReveal
-   Social media integration
-   Downloadable CV option


TECHNOLOGIES USED

Frontend:HTML5
	 CSS3
	 JavaScript (Vanilla JS)

Libraries and Tools:
		    Swiper.js (Testimonial slider) 
		    ScrollReveal.js(Scroll animations) 
		    Boxicons (Icon library) - CDN integration
